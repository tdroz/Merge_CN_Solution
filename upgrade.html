Upgrade instructions

  • Note the name of your install folder as displayed in the config screen
  • Click the Download button
  • Click the 'Open' button in the File Download screen
  • When the Explorer shows the extracted files, then select and copy them all: Ctrl-a, Ctrl-c
  • Go to your install folder
  • Paste all files (Ctrl-v, or right-click and Patse from the menu)
  • Replace all existing files in that folder
  • Reload the Janban board by restarting Outlook
